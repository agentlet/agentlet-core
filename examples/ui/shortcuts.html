Keyboard shortcuts

This example demonstrates the keyboard shortcuts system in Agentlet Core using hotkeys-js.

What this example shows:

  • Default Agentlet shortcuts (Esc for closing dialogs)
  • Optional quick command dialog (Ctrl/Cmd+; - enabled in this example)
  • Custom shortcut registration and management
  • Shortcut scoping and input field handling
  • Visual shortcut help system
  • Dynamic shortcut enabling/disabling

Shortcuts API examples:

Register a Global Help Shortcut

// Register F1 for help (works everywhere, even in inputs)
window.agentlet.utils.shortcuts.register('f1', () => {
    window.agentlet.utils.shortcuts.showHelp();
}, {
    description: 'Show shortcuts help',
    allowInInputs: true
});

Register Multiple Key Combinations

// Register both Ctrl+; and Cmd+; for cross-platform compatibility
window.agentlet.utils.shortcuts.register('ctrl+;,cmd+;', () => {
    window.agentlet.utils.Dialog.commandPrompt({
        title: 'Command Palette',
        placeholder: 'Type a command...'
    }, console.log);
}, {
    description: 'Open command palette',
    allowInInputs: false
});

Panel Control Shortcut

// Register F2 to toggle panel visibility
window.agentlet.utils.shortcuts.register('f2', () => {
    const isVisible = window.agentlet.ui.container.style.display !== 'none';
    if (isVisible) {
        window.agentlet.ui.hide();
    } else {
        window.agentlet.ui.show();
    }
}, {
    description: 'Toggle panel visibility',
    allowInInputs: true  // Works even in input fields
});

Manage Shortcuts

// Check if registered
const isRegistered = window.agentlet.utils.shortcuts.isRegistered('ctrl+j');

// Get all shortcuts
const shortcuts = window.agentlet.utils.shortcuts.getShortcuts();

// Unregister a shortcut
window.agentlet.utils.shortcuts.unregister('ctrl+j');

// Enable/disable all shortcuts
window.agentlet.utils.shortcuts.setEnabled(false);
