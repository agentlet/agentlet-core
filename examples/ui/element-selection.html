Element selection

This example demonstrates interactive element selection capabilities using Agentlet Core's ElementSelector utility.

What this example shows:

  • Interactive element selection with visual feedback
  • CSS selector generation from selected elements
  • Multiple selection modes (single, multiple, filtered)
  • Custom selection callbacks and event handling
  • Element highlighting during selection process
  • Selector validation and optimization

Element selection API examples:

Basic Element Selection

// Start element selection
window.agentlet.utils.ElementSelector.start((element, elementInfo) => {
    console.log('Selected element:', element);
    console.log('Generated selector:', elementInfo.cssSelector);
    console.log('Element details:', elementInfo);
});

// Stop active selection
window.agentlet.utils.ElementSelector.stop();

Filtered Selection

// Select only buttons
window.agentlet.utils.ElementSelector.start((element, elementInfo) => {
    console.log('Selected button:', element);
    console.log('Button selector:', elementInfo.cssSelector);
}, {
    selector: 'button',
    message: 'Click on any button element'
});

// Custom filtering with validation
window.agentlet.utils.ElementSelector.start((element, elementInfo) => {
    if (element.classList.contains('my-class')) {
        console.log('Valid element selected:', elementInfo.cssSelector);
    } else {
        console.log('Element does not match criteria');
    }
}, {
    message: 'Click on elements with "my-class" class'
});

Advanced Selection Options

// Selection with custom message and filtering
window.agentlet.utils.ElementSelector.start((element, elementInfo) => {
    console.log('Element selected:', element);
    console.log('CSS selector:', elementInfo.cssSelector);
    console.log('XPath:', elementInfo.xpath);
    console.log('Element info:', elementInfo);
}, {
    selector: 'input, button',
    message: 'Click on input fields or buttons only'
});

// Get comprehensive element information
const element = document.getElementById('my-element');
const elementInfo = window.agentlet.utils.ElementSelector.getElementInfo(element);
console.log('Element details:', elementInfo);
console.log('Optimized selector:', elementInfo.cssSelector);
