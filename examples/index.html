Agentlet core examples

Examples demonstrating core functionality and features of the Agentlet framework.

Getting started: Run npm run demo in the project root to build the library and start this dev server. Examples won't work with file:// protocol and require a web server.

Examples by category

basics/

Core concepts and fundamental usage patterns

  • hello-world.html - Basic initialization and setup
  • panel-basics.html - Panel operations and events
  • simple-module.html - Creating custom modules

ui/

User interface components and interactions

  • dialogs.html - All dialog types (info, input, custom, fullscreen)
  • shortcuts.html - Keyboard shortcuts and hotkeys
  • highlighting.html - Element highlighting and overlays
  • message-bubbles.html - Toast notifications (coming soon)
  • element-selector.html - Interactive element selection (coming soon)

forms/

Form extraction, filling, and automation

  • extraction.html - Extract form structure and data
  • filling.html - Automated form filling with AI support

tables/

Table data extraction and export capabilities

  • extraction.html - Extract table data with pagination
  • excel-export.html - Export to Excel with SheetJS

ai/

AI integration and automation capabilities

  • text-prompts.html - OpenAI integration with text, images, and PDFs
  • multimodal.html - Advanced multimodal AI examples (coming soon)
  • pdf-analysis.html - Dedicated PDF processing demo (coming soon)

storage/

Data persistence and environment management

  • environment.html - Environment variables management (coming soon)
  • cookies.html - Cookie handling and persistence (coming soon)
  • local-storage.html - Local storage utilities (coming soon)

auth/

Authentication and authorization flows

  • oauth-popup.html - OAuth popup-based authentication (coming soon)
