üìñ Tips for Using Examples

  • Open developer console: Many examples log detailed information to the console
  • Check network tab: Watch for agentlet core loading in the network tab
  • Inspect elements: Use browser dev tools to examine generated HTML structure
  • Try keyboard shortcuts: Many dialogs support escape key for closing
  • Consistent structure: Examples use standardized console and status patterns

üîß Prerequisites

Before running examples, ensure you have:

  • Built the agentlet core library: npm run build (or use npm run demo to do this automatically)
  • A local web server (examples won't work with file:// protocol) - this is done automatically when you run npm run demo
  • Modern browser with JavaScript enabled
  • For AI examples: OpenAI API key (set in the demo interface)
