💬 Info Dialogs - Agentlet Core

This example demonstrates the various types of information dialogs available in Agentlet Core.

What this example shows:

  • Basic info dialogs (info, success, warning, error)
  • Custom dialogs with HTML content
  • Dialog callbacks and button handling
  • Dialog styling and customization options
  • NEW: Fullscreen dialogs for large content

📜 Dialog API Examples:

Basic Dialog Types

// Simple info dialog
window.agentlet.utils.Dialog.info('Message', 'Title');

// Success dialog
window.agentlet.utils.Dialog.success('Success message', 'Success');

// Warning dialog
window.agentlet.utils.Dialog.warning('Warning message', 'Warning');

// Error dialog
window.agentlet.utils.Dialog.error('Error message', 'Error');

Custom Dialog with Options

// Advanced dialog with custom buttons and callback
window.agentlet.utils.Dialog.show('info', {
    title: 'Custom Dialog',
    message: 'Choose an action:',
    icon: '🔘',
    buttons: [
        { text: 'Cancel', value: 'cancel', secondary: true },
        { text: 'Save', value: 'save', primary: true, icon: '💾' }
    ],
    allowHtml: true
}, (result) => {
    console.log('User clicked:', result);
});

Fullscreen Dialog

// Fullscreen dialog for large content
window.agentlet.utils.Dialog.fullscreen({
    title: 'Large Content Display',
    message: '<h3>Large HTML Content</h3><p>Tables, forms, documentation...</p>',
    icon: '🖥️',
    allowHtml: true,
    scrollable: true,
    closeOnOverlay: true,
    buttons: [
        { text: 'Save', value: 'save', primary: true },
        { text: 'Close', value: 'close' }
    ]
}, (result) => {
    console.log('Fullscreen result:', result);
});
